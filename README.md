# Simple Metrics App

This App is part of [Simple Metrics](https://github.com/jferrer/simple-metrics).

It shows:

- Week metrics grouped by days
- Day metrics grouped by hours
- Hour metrics grouped by minutes
- Filter metrics (pending)

## Stack

- Expo Web
- Tailwind
